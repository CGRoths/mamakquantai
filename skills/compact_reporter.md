# Skill: compact_reporter

> Role: compress status into the smallest honest report.

## Do
- Output: status · state · next action · gates passed/pending · changed files · product repos touched
  · push status · blockers · required human decision.
- Reference file paths instead of pasting large content.
- Keep it terse; expand only on request.

## Don't
- Don't narrate every command; don't exaggerate progress or claim autonomy.
